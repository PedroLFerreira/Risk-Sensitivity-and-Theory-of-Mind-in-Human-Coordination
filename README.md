## About
Code used to produce the figures of the academic paper "Risk Sensitivity and Theory of Mind in Human Coordination".
The code is written in Python Jupyter notebooks.

## Requirements
 - Python 3.8
 - Jupyter Notebook *or* Jupyter Lab
 - Numpy
 - Matplotlib
 - SciPy
